⚠️ Vigtig information om databehandling

KommuNemt bruger OpenAI GPT-4o (amerikansk virksomhed) til at analysere dine breve. For at forklare dit brev skal AI'en kunne læse det fulde indhold — inklusive eventuelle personoplysninger. Vi er ærlige om dette, fordi gennemsigtighed er vigtigere end markedsføring.

1. Sådan virker det — helt ærligt

Når du tager et billede af dit brev:

  • Trin 1: Billedet sendes via HTTPS (transportkryptering) fra din telefon til vores server.
  • Trin 2: Vores server sender billedet videre til OpenAI's API for analyse. OpenAI's servere er i USA.
  • Trin 3: AI'en læser dit brev og sender en forklaring tilbage.
  • Trin 4: Vi sletter billedet straks. Intet gemmes på vores servere.

2. Hvad betyder "krypteret"?

Når vi siger data sendes krypteret, mener vi transportkryptering (HTTPS/TLS). Det betyder at data er beskyttet under overførslen — på samme måde som din netbank. Det betyder IKKE at data er uforståelig for OpenAI — AI'en skal kunne læse brevet for at forklare det.

3. OpenAI som tredjepart

Vi bruger OpenAI's API med følgende sikkerhedsgarantier:

  • Zero Data Retention: OpenAI gemmer IKKE data fra API-kald.
  • Ingen træning: Dine breve bruges IKKE til at træne OpenAI's modeller.
  • SOC 2 Type II certificeret: OpenAI er uafhængigt revideret for datasikkerhed.
  • Data Processing Agreement (DPA): OpenAI tilbyder en GDPR-kompatibel databehandleraftale.

Men vi er ærlige: Dit brev passerer OpenAI's servere i USA. Selvom de har Zero Data Retention, er det stadig en tredjepart. Du bør overveje dette før du sender meget følsomme dokumenter.

4. Hvad vi IKKE kan gøre

Nogle spørger: "Kan I ikke anonymisere brevet før det sendes til AI?"

Nej — og her er hvorfor: For at anonymisere et brev ville vi have brug for en anden AI til at læse brevet først. Så ville data allerede være sendt til en AI-tjeneste.

I stedet fokuserer vi på: straks sletning og brug af en API-leverandør med Zero Data Retention.

5. Hvad vi GØR for at beskytte dig

  • Brevbilleder slettes straks efter analysen — vi gemmer INTET
  • Vi bruger kun OpenAI's API (Zero Data Retention) — ikke ChatGPT
  • Vi sender kun billedet til OpenAI — aldrig dit navn, enheds-ID eller betalingsinfo
  • Analyseresultater gemmes kun lokalt på din telefon
  • Betalinger håndteres af Stripe (PCI DSS-certificeret)

6. Perspektiv: Alternativerne

Mange danskere deler i dag deres offentlige breve på Facebook-grupper, sender dem via email, eller uploader dem til online-oversættere. Alle disse metoder involverer tredjeparter der gemmer dine data. KommuNemt sletter dine data straks.

7. Hvilke data gemmer vi?

  • Enheds-ID: Et anonymt ID til at holde styr på dine kreditter.
  • Betalingshistorik: Transaction-ID fra Stripe (ingen kortdata).
  • Sprogpræference: Gemmes lokalt på din enhed.

Vi gemmer IKKE: Billeder, breve, analyseresultater, navne, adresser eller CPR-numre.

8. Dine rettigheder (GDPR)

  • Ret til indsigt i dine data
  • Ret til sletning
  • Ret til berigtigelse
  • Ret til dataportabilitet
  • Ret til at klage til Datatilsynet

9. EU AI Act

  • Alle analyser er tydeligt mærket som AI-genererede
  • Analyserne er vejledende og udgør IKKE juridisk rådgivning
  • Søg altid professionel rådgivning
